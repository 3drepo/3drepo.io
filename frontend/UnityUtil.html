Enable measuring tool. This will allow you to start measuring by clicking on the model

getObjectsStatus(account, model, promise)

Get Object Status within the viewer. This will return you the list of objects that are currently set invisible, and a list of object that are currently highlighted. The object status will be returned via the promise provided.
Parameters:
Name Type Description
account string name of teamspace
model string name of the model
promise object promise that the function will resolve with the object status info.

highlightObjects(account, model, idArr, color, toggleMode)

Highlight objects
Parameters:
Name Type Description
account string name of teamspace
model string name of model
idArr Array.<string> array of unique IDs associated with the objects to highlight
color Array.<number> RGB value of the highlighting colour
toggleMode bool If set to true, existing highlighted objects will stay highlighted. Also any objects that are already highlighted will be unhighlighted

setViewpoint(pos, up, forward, lookAt, accountopt, modelopt)

Move viewpoint to the specified paramters teamspace and model is only needed if the viewpoint is relative to a model
Parameters:
Name Type Attributes Description
pos Array.<number> 3D point in space where the camera should be
up Array.<number> Up vector
forward Array.<number> forward vector
lookAt Array.<number> point in space the camera is looking at. (pivot point)
account string <optional>
 name of teamspace
model string <optional>
 name of model

toggleStats()

Toggle on/off rendering statistics. When it is toggled on, list of stats will be displayed in the top left corner of the viewer.

Toggle visibility of the given list of objects

Name Type Description

model string name of model
ids Array.<string> list of unique ids to toggle visibility
visibility bool true = toggle visible, false = toggle invisible

updateClippingPlanes(clipPlane, requireBroadcast, accountopt, modelopt)

Update the clipping plane to the given direction teamspace and model is only needed if the viewpoint is relative to a model
Parameters:
Name Type Attributes Description
clipPlane Object object containing the clipping plane
requireBroadcast bool if set to true, UnityUtil.clipBroadcast will be called after it is set.
account string <optional>
 name of teamspace
model string <optional>
 name of model
Example
//Clipping plane is defined by the plane normal, distance from origin and it's direction
//direction = -1 means it will clip anything above the plane, 1 otherwise.
UnityUtil.updateClippingPlanes({normal : [0,-1,0], distance: 10, clipDirection: -1}, false)
